# Suspicious Activity Detection Using AI and Drones


# Overview
This project detects suspicious activities using YOLOv8 and computer vision on drone imagery.


# Features
- Object Detection
- Bounding Box Detection
- YOLOv8 Model
- Google Colab Compatible


# Dataset
Figshare Suspicious Activity Recognition Dataset


# Tools
- Python
- YOLOv8
- OpenCV
- Google Colab


# How to Run
1. Install requirements
2. Upload dataset
3. Run notebook
