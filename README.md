# Suspicious Activity Recognition Using Deep Learning

## 📌 Overview
This project focuses on detecting suspicious human activities from images using deep learning techniques. The system classifies activities such as falling, fighting, firing, running, and fire-related events to improve security and surveillance efficiency.

Unlike traditional video-based approaches, this project uses image-based models to reduce computational complexity while maintaining high accuracy.

---

## 🎯 Features
- Image-based Human Activity Recognition
- Multiple Deep Learning Models:
  - CNN (Baseline Model)
  - VGG19
  - ResNet50
  - Xception
- Automated Dataset Splitting (Train/Test/Validation)
- Performance Evaluation:
  - Accuracy & Loss Curves
  - Confusion Matrix
  - Classification Report
- Model Comparison

---

## 📂 Dataset
- **Source:** Figshare Suspicious Activity Recognition Dataset  
- **Classes:** Falling, Fighting, Fire, Firing, Running
